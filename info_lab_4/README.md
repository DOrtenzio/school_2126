# 💻 Laboratorio di Informatica - Quarta Superiore

Benvenuto nella repository che raccoglie tutte le esercitazioni e i progetti sviluppati durante le ore di **Laboratorio di Informatica** della classe quarta. Questo spazio traccia il percorso di apprendimento, focalizzandosi sullo sviluppo web frontend, logica in JavaScript e programmazione pratica.

## 📂 Struttura della Repository

La repository è organizzata in cartelle dedicate per ogni specifico argomento o progetto:

### 🚀 Applicazioni e Progetti JavaScript
*   `js-cv` - Curriculum Vitae interattivo sviluppato in JavaScript.
*   `js-memory` - Il classico gioco del Memory per allenare la logica sui cicli e array.
*   `js-timer` - Applicazione di cronometro / timer con gestione degli intervalli di tempo.

### 📝 Esercitazioni Pratiche (Anno Scolastico 2024)
*   `pratica1-24` - Introduzione e primi script di base.
*   `pratica2-24` - Strutture condizionali e manipolazione dati primaria.
*   `pratica3-24` / `pratica3.1-24` - Esercizi sui cicli e prime iterazioni complesse.
*   `pratica4-24` - Gestione delle funzioni e modularità del codice.
*   `pratica5-24` - Manipolazione avanzata di stringhe e array.
*   `pratica6-24` - Primi approcci con il DOM e interazione utente.

### 📐 Esercitazioni Pratiche (Anno Scolastico 2025)
*   `pratica8-25` - Gestione degli eventi avanzata nel browser.
*   `pratica9-25` - Convalida dei form e interazione con l'utente.
*   `pratica10-25` - Creazione dinamica di elementi HTML tramite JS.
*   `pratica11-25` - Approfondimento sugli oggetti e strutture dati complesse.
*   `pratica12-25` - Utilizzo delle API del browser e LocalStorage.
*   `pratica13-25` - Chiamate asincrone e gestione delle Promesse / Fetch API.
*   `pratica14-25` - Progetto di sintesi e architettura del codice frontend.
*   `pratica15-25` - Ottimizzazione, debugging e rifinitura finale dei progetti.

## 🛠️ Tecnologie Utilizzate

*   **HTML5** - Strutturazione delle pagine web.
*   **CSS3** - Stile, layout responsivi e animazioni.
*   **JavaScript (ES6+)** - Logica di programmazione, manipolazione del DOM e asincronia.

## 💻 Come Eseguire i Progetti

1. **Clona la repository** sul tuo computer locale:
   ```bash
   git clone https://github.comDOrtenzio/info_lab_4.git
   ```
2. **Entra nella cartella** del progetto che vuoi testare:
   ```bash
   cd info_lab_4
   ```
3. **Apri il file `index.html`** nel tuo browser preferito per vedere l'applicazione in funzione.
